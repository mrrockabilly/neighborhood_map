# Neighborhood Map

A map of popular ice cream shops in Los Angeles.


https://mrrockabilly.github.io/neighborhood_map/

## Technologies Used

Knockout.JS, Google and Yelp APIs.

## Running Locally

```
git clone https://github.com/mrrockabilly/neighborhood_map
cd neighborhood_map
open ./index.html
```
